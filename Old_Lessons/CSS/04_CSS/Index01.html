Форматы файлов.

.DOC или .doc (от английского документ) - расширение файла, используемого для файлов, представляющих текст, с разметкой или без. Расширение .DOC часто использовалсь для обозначения простых текстовых файлов без форматирования, однако позже стало использоваться для доичных форматов с разметкой.
Документ MS Word

Portable Document Format (PDF) - межплатформенный формат электронных документов, разработанный фирмой Adobe Systems с использованием ряда воможностей языка Postscript.
Документ PDF

ZIP - попудярный формат сжатия данных и архивации файлов. Файл в этом формате обычно имеет расширение PDF и хранит в сжатом или несжатом виде один или несколько файлов, которые можно из него извлечьпутем распаковки с помощью специальной программы.
